# Renev Medication

**Get features without downloading an update, always the latest data in the app, and get a notification when an update is available.**

Renev Medication has quick buttons in the settings to the Discord server and send an email to the developer, as well as links to programs where you can support the project. You also receive a notification when there is an update, simple and effortless.

## Key Features

* **No expired connections:** How annoying would it be if a connection had expired and you were subsequently unable to make contact or support the project? We eliminate this by updating the connections.
* **Quickly find out when an update is available:** You will receive a notification that an update is available.
* **Block specific versions:** What do we do if there has been an update containing many bugs? A simple notification is not sufficient in that case; blocking that version might sound bad, but it ensures that the user experience remains good.

# What I promise

I will never add advertisements to the app; this only makes the user experience worse. Your data will always remain secure and will never reach the Renev Medication servers. If we were to add accounts, the data would not be stored with us either.
